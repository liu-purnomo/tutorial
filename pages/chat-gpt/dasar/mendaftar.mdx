# Membuat Akun di Open AI

import { Screenshot } from "../../../components"
import loginPage from "../../../public/images/chat-gpt/login-page.png"
import startChat from "../../../public/images/chat-gpt/start-chat.png"

1. Pastikan sudah memiliki email yang aktif

2. Buka Website [Open AI](https://chat.openai.com/auth/login)

<Screenshot src={loginPage} alt="Home Page" />

3. Jika Anda sudah punya akun OpenAI, klik tombol `Log In`. Jika Anda belum punya akun OpenAI, klik tombol `Sign Up`

4. Setelah masuk ke ChatGPT, Anda akan menemukan halaman seperti di bawah ini. Anda bisa memasukkan perintah pada kotak teks yang berada di bagian bawah dan ChatGPT akan menjawab pertanyaan Anda.

<Screenshot src={startChat} alt="Start Chat" />
